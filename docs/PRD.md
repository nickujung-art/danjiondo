# PRD: 단지온도 (danjiondo)

## 목표
창원·김해 회원이 **시세 조회와 이웃 의견 수집을 한 페이지에서 끝내는** 부동산 실거래가 사이트. 외부 데이터 서비스(호갱노노)와 카페 커뮤니티 사이의 왕복 흐름을 단일 흐름으로 통합.

## 핵심 가치 명제
> "창원·김해의 모든 단지를 — 실거래가, 학군, 관리비, 편의시설, 그리고 우리 동네 의견까지 — 한 화면에서."

## 사용자

### Persona A: 실수요자 / 갈아타기 검토자 (주력)
- 30~50대, 창원 거주. 본인 실거주 + 자녀 학군 동시 고려.
- 카페에서 "OO 단지 어떤가요?" 질문 후 답변 1~2일 대기 중.
- **JTBD**: "이 단지 사도 되는지, 데이터와 동네 사람들 의견을 같이 보고 30분 안에 결정 짓고 싶다."

### Persona B: 투자자 / 임장러
- 30~40대, 카페 적극 회원. 신고가 갱신·거래 급증 신호에 민감.
- **JTBD**: "관심 단지 신고가 갱신 즉시 알고, 이번 주 임장 동선을 효율화하고 싶다."

### Persona C: 분양 검토자 / 신혼·청년
- 20~30대, 김해 신혼 세대. 회원 가입은 "혜택 있을 때만". 비회원 SEO 유입 다수.
- **JTBD**: "예정 분양가가 주변 단지 대비 싼지 비싼지 30초 안에 판단하고 싶다."

### Persona D (광고주): 지역 분양사 / 중개사
- 카페 광고 이미 집행 중. 사이트 광고는 추가 채널.
- **JTBD**: "타깃 회원에게 도달했음을 노출/클릭 수치로 광고주 본사에 보고 가능해야 한다."

## 핵심 사용자 시나리오

### S1. 카페 → 사이트 첫 방문
1. 카페 공지글 링크 클릭 → 단지온도 랜딩 (비공개 베타 기간: 카페 공지 단일 유입 채널)
2. "오늘 신고가 카드" — 익숙한 단지 등장 → 체류
3. 신고가 카드 클릭 → 단지 상세 → 10년 그래프 → 관리비 → 학군
4. 즐겨찾기·알림 유도는 단지 상세에서만 노출

### S2. 관심 단지 추적
1. 회원이 단지 즐겨찾기 → 신고가 갱신 감지 → 알림 큐 enqueue
2. 이메일 + 웹 푸시 발송 → 회원 클릭 → 단지 상세

### S3. 비회원 SEO 유입
1. 검색엔진 "김해 율하 OO 시세" → 단지 상세 (SSG/ISR)
2. 인근 신규 분양 광고 노출
3. "분양 알림 받기" CTA → 이메일 입력 → 회원 전환

### S4. 지도 임장 (Persona B)
1. 모바일 지도 페이지 → 핀별 평당가 라벨
2. 핀 클릭 → 단지 미리보기 카드 → 단지 상세

### S5. 광고주 캠페인 등록
1. 광고주 어드민 로그인 → 캠페인 등록 → 운영자 검수 → 승인 → 게재
2. 노출/클릭 통계 매일 확인

## 핵심 기능 (V1.0 — 출시 완료)
1. **단지 검색** — 자동완성 + 슬러그 URL, FTS + pg_trgm, complex_aliases 학습
2. **단지 상세** — 10년 Recharts 그래프, 거래 내역(취소·정정 제외), 시설 섹션
3. **지도 검색** — 카카오맵, 평당가 핀 라벨, supercluster 클러스터링
4. **학군** — 학교알리미 322개교 (학급당학생수·진학률·연락처·순위), 진학률 중/고 분리
5. **관리비** — K-apt 연동, 단지 상세 표시
6. **분양 정보** — crawl-presale.ts + Gemini HTML 파서, presale_enriched 테이블
7. **AI 코멘트** — Chronos 12개월 예측 + Groq 코멘터리 (964건)
8. **투자 분석** — Gap 분석, 사분면 분석, 지역 인구 시각화
9. **단지 비교** — 최대 3개 단지 비교
10. **회원 / 즐겨찾기 / 알림** — Supabase Auth (Naver OAuth + 이메일 OTP), 웹 푸시
11. **광고** — 분양+중개 캠페인, 어드민 검수, 노출·클릭 통계
12. **PWA** — Serwist, 웹 푸시 (iOS 16.4+), 오프라인 대응
13. **재건축 정보** — 어드민 수동 입력, 단지 상세 표시
14. **랜딩** — 신고가 카드 + 랭킹 4종 탭
15. **어드민 콘솔** — 회원·광고·카드뉴스·공인중개사·분양·재건축·신고 관리
