# 창원부동산랩 — 인스타그램 카드뉴스 디자인 사양서 (CARDDESIGN.md)

> 클로드코드(또는 다른 개발 환경)에서 이 문서만 보고 카드뉴스를 그대로 재현·양산할 수 있도록 만든 사양서입니다.
> 확정 방향: **표지 B (화이트 클린)**. 본 문서의 모든 색/타이포/레이아웃 토큰은 이 방향 기준입니다.

## 1. 기본 규격

| 항목 | 값 |
|------|-----|
| 캔버스 | **1080 × 1080px** (인스타 정사각형) |
| 카드 장수 | 4장 (표지 + 하이라이트 + 전체순위 + 클로징) |
| 폰트 | **Pretendard** (900/800/700/600/500 사용) |
| 안전 여백 | 좌우 상하 **80px** 그리드 |
| 단위 표기 | 가격은 `만원`. 예) `16억 4,000` (= 16억 4,000만원) |
| 출처 | 국토교통부 실거래가 공개시스템 |

## 2. 컬러 토큰

로고(파랑+빨강) 기반의 절제된 팔레트. 파랑은 **포인트로만**, 나머지는 무채색.

| 토큰 | HEX | 용도 |
|------|-----|------|
| `--brand` | `#0066FF` | 강조, 링크, 1차 CTA, 포인트 텍스트 |
| `--brand-tint` | `#EAF2FE` | 표지 거대 고스트 숫자, 옅은 배경 |
| `--ink` | `#152038` | 주요 텍스트(네이비), 클로징 배경 |
| `--ink-2` | `#5B6677` | 보조 설명 텍스트 |
| `--ink-3` | `#8A93A3` | 캡션/메타 |
| `--gold` | `#FFC93C` | 1위 강조, 네이비 위 하이라이트 |
| `--gold-2` | `#FFAB00` | 순위 표 1위 숫자 |
| `--surface` | `#FFFFFF` | 카드 기본 배경 |
| `--surface-2` | `#F7F8FA` | 본문 카드 배경 |
| `--line` | `rgba(112,115,124,0.18)` | 표 구분선(헤어라인) |
| `--placeholder` | `#C4CAD3` | 미입력 데이터 회색 |

**규칙**: 파랑/골드는 강조 전용. 빨강·초록은 사용하지 않음(상태 표시 외). 그라데이션 금지.

## 4. 카드별 레이아웃

### 4-1. 표지 (Cover B · 화이트 클린)
- 배경 `#FFFFFF`, 상단에 **높이 14px `--brand` 바**(전체 너비).
- 좌상단 브랜드 락업: 로고 이미지(46px, radius 10) + `창원부동산랩`(30px/700).
- 에이브로우: `WEEKLY REPORT · 2026.06 W1` (28px/700, `--brand`).
- 대제목 3줄: `창원 성산구` / `84㎡ {실거래가:파랑}` / `랭킹 TOP 10` (104px/900).
- **거대 고스트 숫자 `10`**: 우하단, 560px/900, 색 `--brand-tint`, `z-index:0` (제목 뒤).
- 하단 카피(z-index 1): `이번 주 가장 비싸게 거래된 / 국민평형 아파트는 어디일까요?`
- 최하단 위치 메타: 핀 아이콘(파랑) + `경남 창원시 성산구 · 전용 84㎡ 기준`.

### 4-2. 2p — TOP 3 하이라이트
- 배경 `--surface-2(#F7F8FA)`.
- 헤더: `HIGHLIGHT`(파랑) + `최고가 거래 TOP 3`(64px/900).
- **1위 카드**: 배경 `--ink(네이비)`, radius 28, 흰 글씨. 순위뱃지=골드 원(96px), 가격=골드.
- **2·3위 카드**: 흰 배경 + `--line` 보더, radius 28. 순위뱃지=연회색 원(84px).
- 각 행 구성: `[순위 원] [단지명 + 성산구·전용84㎡] ────── [가격 + 만원]` (가격 우측정렬).
- 하단 캡션: 출처/기간.

### 4-3. 3p — 전체 순위 1~10위 (표)
- 배경 `#FFFFFF`.
- 헤더: `FULL RANKING`(파랑) + `실거래가 순위 1~10위`(60px/900).
- 10개 행, 각 행: `[순위숫자 54px폭] [단지명 flex] [가격 우측]`, 행 하단 `--line` 헤어라인.
- 1위 숫자 색 `--gold-2`, 2~10위 숫자 색 `#9AA4B2`.
- **미입력 행(4~10위)은 `--placeholder(#C4CAD3)` 회색**으로 `단지명 입력` / `0억 0,000` 표기.
- 하단 캡션: `단위: 만원 · 4~10위는 실데이터로 교체`.

### 4-4. 4p — 클로징 / CTA
- 배경 `--ink(네이비)`, 흰 글씨.
- 상단 브랜드 락업(로고 56px + 창원부동산랩 34px).
- H2: `매주 업데이트되는 / 창원 {실거래가 리포트:골드}` (80px/900).
- 설명: `우리 동네 아파트 시세가 궁금하다면 / 창원부동산랩을 팔로우하세요.`
- CTA 2개(pill): `팔로우 +`(배경 파랑) / `저장하기`(흰 보더 아웃라인).
- 최하단 면책 문구(헤어라인 위, `--ink-3` 톤).

## 5. 데이터 모델

```json
{
  "region": "창원시 성산구",
  "area": "84㎡",
  "week": "2026년 6월 1주차",
  "source": "국토교통부 실거래가 공개시스템",
  "period": "2026.06.01 ~ 06.07 신고 건",
  "ranking": [
    { "rank": 1, "name": "용지아이파크",            "price": "16억 4,000" },
    { "rank": 2, "name": "용지더샵레이크파크",       "price": "15억 7,000" },
    { "rank": 3, "name": "창원 중동 유니시티 2단지",  "price": "10억 8,500" },
    { "rank": 4, "name": "",  "price": "" },
    { "rank": 5, "name": "",  "price": "" },
    { "rank": 6, "name": "",  "price": "" },
    { "rank": 7, "name": "",  "price": "" },
    { "rank": 8, "name": "",  "price": "" },
    { "rank": 9, "name": "",  "price": "" },
    { "rank": 10, "name": "", "price": "" }
  ]
}
```
> `name`/`price`가 빈 값이면 회색 플레이스홀더(`단지명 입력` / `0억 0,000`)로 렌더링.
> **확인 필요**: 3위 원문은 `10억 8,5000` → `10억 8,500만원`으로 해석함.

## 6. 주차별 재사용 방법

매주 카드뉴스를 새로 만들 때:
1. `data.ranking` 10개 채우기(없으면 빈 값 → 자동 회색).
2. `region` / `area` / `week` / `period` 텍스트만 교체.
3. 표지 거대 숫자(`10`), 제목의 `TOP 10`은 고정.
4. 1위 가격·단지명이 바뀌면 2p의 골드 1위 카드도 자동 반영.

## 8. 디자인 원칙 (요약)

- **정보 중심**: 숫자가 주인공. 큰 굵은 숫자 + 명확한 순위.
- **절제된 색**: 흰/네이비 베이스, 파랑·골드는 포인트로만.
- **여백**: 80px 그리드 엄수, 카드 radius 24~28.
- **헤어라인**: 구분선은 불투명 회색 대신 `rgba(112,115,124,0.18)`.
- **그라데이션·이모지·빨강/초록 장식 금지.**
- 모든 UI 문구는 문장형 소문자/한글 자연스럽게, 과장된 카피 지양.
