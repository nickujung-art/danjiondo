|-------|-------|----------|-------|
| 2xs | 4px | `p-1` / `gap-1` | Icon internal gaps, badge padding |
| xs | 8px | `p-2` / `gap-2` | Compact element spacing, chip gaps |
| sm | 12px | `p-3` / `gap-3` | Card internal sub-element spacing |
| md | 16px | `p-4` / `gap-4` | Content horizontal padding (`px-4`), default element spacing |
| lg | 24px | `p-6` / `gap-6` | Card internal padding, section gaps |
| xl | 32px | `p-8` / `gap-8` | Section separators on desktop |

**Exceptions:**
- Bottom tab bar height: **64px** (fixed) + `env(safe-area-inset-bottom)` for iOS notch

- BottomSheet top radius: **20px** (`rounded-t-[20px]`)
- BottomSheet drag handle: 40px × 4px, `rounded-full`, `bg-[var(--line-default)]`

- **z-index hierarchy:** Page content (0) → BottomTabBar (z-40) → BottomSheet overlay (z-200) → BottomSheet content (z-201)

```
    <AppHeader />          ← sticky top-0

```
- **viewport:** must include `viewportFit: 'cover'` in Next.js `Viewport` export (enables `env(safe-area-inset-bottom)` on iOS)

### Content Width (D-09)
- **Mobile (default):** `w-full px-4` (100% width, 16px horizontal padding)
- **Desktop override:** `sm:max-w-3xl sm:mx-auto` OR `sm:max-w-screen-lg sm:mx-auto` depending on page
- **Pattern:** `<main className="px-4 py-6 sm:px-8 sm:py-8 max-w-screen-xl mx-auto">`

### Tailwind Responsive Rule (D-06)
- Default (no prefix) = mobile layout
- `sm:` (640px+) = tablet/desktop override
- **FORBIDDEN:** `max-sm:` pattern. Never use this.

## Interaction Contract

### Touch Targets
- **Rule:** Every button, link, chip, and tab must have `min-h-[44px] min-w-[44px]` — **no exceptions**
- **Chips that are too small:** Increase padding or group with neighboring chips to meet 44px
- **Tab bar items:** Each tab is `flex-1` in a 64px-tall nav — minimum height satisfied by container height

### BottomSheet Interactions
- **Open trigger:** Button tap (explicit — no swipe-from-edge to open)
- **Dismiss:** Drag down past snap threshold OR tap overlay
- **Focus trap:** Vaul handles automatically (WCAG 2.4.3 compliant)
- **Scroll:** `overflow-y: auto` inside content div; outer container does NOT scroll
- **Max height:** `90dvh` — does not cover full screen

### Tab Swipe (단지상세)
- **Gesture:** Horizontal swipe on tab content area only (D-13)
- **Tabs to swipe:** DealTypeTabs content panels (거래내역 / 시설 / 관리비 / AI코멘트 etc.)
- **URL sync:** Swipe completion updates nuqs `tab` searchParam. Direct URL navigation sets Embla `startIndex`.
- **Tap on tab label:** Also navigates (existing behavior retained)

### Animation Rules (CLAUDE.md)
- **Allowed compositor properties only:** `transform`, `opacity`, `clip-path`
- **FORBIDDEN layout properties in animations:** `width`, `height`, `top`, `margin`, `padding`, `bottom`
- **BottomSheet open/close:** `transform: translateY()` + `opacity` (Vaul handles this internally)
- **Tab swipe transition:** Embla handles via `transform: translateX()` on the flex track
- **Hover/active transitions:** `transition: 120ms ease` — buttons, tabs, chips
- **Fade-in:** `opacity 0→1, 0.2s ease-out` (page load, card render)
- **Slide-up:** `translateY(8px→0) + fade-in, 0.3s ease-out` (BottomSheet open alternative)
- **FORBIDDEN:** `backdrop-filter: blur()`, gradient text, glow `box-shadow` animation, gradient orb

### Safe Area (iOS)
- **Requirement:** `viewport` export must have `viewportFit: 'cover'`
- **BottomTabBar:** `paddingBottom: env(safe-area-inset-bottom, 0px)` via inline style
- **BottomSheet:** `paddingBottom: env(safe-area-inset-bottom, 20px)` via inline style
- **body:** `pb-[calc(64px+env(safe-area-inset-bottom,0px))]` — Tailwind arbitrary value

### html2canvas Capture (공유 캡처)
- **Rule:** `data-capture-hide="true"` on `AppHeader` and `BottomTabBar`
- **Existing `ShareButton.tsx` already removes** `[data-capture-hide]` elements in `onclone` callback
- No additional work needed beyond adding the attribute
