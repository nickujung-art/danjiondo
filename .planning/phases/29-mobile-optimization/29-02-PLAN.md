기존 chip() 함수가 inline CSSProperties 객체를 반환하는 패턴을 Tailwind + inline style(색상만)로 전환.

```tsx

function chip(active: boolean): React.CSSProperties {
  return {
    display: 'inline-block', padding: '6px 12px', borderRadius: 20,
    font: '600 12px/1 var(--font-sans)', textDecoration: 'none',
    background: active ? 'var(--dj-orange)' : 'var(--bg-surface-2)',
    color: active ? '#fff' : 'var(--fg-sec)',
    // ...
  }
}

// After — chip() 함수 제거 후 Link/button 에 직접 className + style:
// className에 레이아웃/타이포 → Tailwind
// style에 색상(CSS 변수) → inline
```

각 필터 칩 Link/button 요소에 적용할 패턴:
```tsx
<Link
  href={...}
  className="inline-flex items-center px-3 rounded-full text-xs font-bold whitespace-nowrap shrink-0 min-h-[44px]"
  style={{
    background: active ? 'var(--dj-orange)' : 'var(--bg-surface-2)',
    color: active ? '#fff' : 'var(--fg-sec)',
    textDecoration: 'none',
  }}
>
  {label}
</Link>
```
