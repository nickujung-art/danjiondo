창원부동산이야기
공식 미디어, 창부레터

네이버카페 창원부동산이야기(20만 회원)의 콘텐츠를 웹·SNS로 확장하는 공식 미디어 플랫폼. 카드뉴스·기획기사·실거래가 인포그래픽으로 창원·김해 부동산 이야기를 전합니다.

상태논의 중
01 · 문제 정의

어쩌다 백화점이 됐나

처음의 아이디어는 명확했습니다. 그런데 호갱노노·리치고·아파트미·네이버 부동산을 벤치마킹하면서 기능을 하나씩 추가하다 보니, 어느 순간 모든 것이 다 있지만 아무것도 특별하지 않은 서비스가 됐습니다.

⚠ 지금의 문제

실거래가, 투자분석, AI예측, 학군, 관리비, 학원추천, 카드뉴스, 단지비교, 랭킹… 전국 서비스를 창원/김해 버전으로 축소 복사한 형태. 타겟이 없어서 누구를 위한 서비스인지 불명확.

→
✓ 우리가 가야 할 곳

창원·김해 부동산에 관심 있는 사람이 자연스럽게 소비하고 싶은 콘텐츠를 만드는 미디어. 데이터는 콘텐츠의 배경이 되고, 콘텐츠가 유입을 만든다.

🎯
타겟 부재
부동산을 살 사람, 투자할 사람, 그냥 궁금한 사람 모두를 위한 기능이 섞여있어 어느 누구에게도 특별하지 않음.
🚪
진입 장벽
검색창 중심의 홈은 "뭘 찾아야 하는지 아는 사람"만 쓸 수 있음. 막연히 부동산에 관심 있는 사람은 진입 불가.
🔄
재방문 이유 없음
지금 집 살 생각이 없으면 다시 올 이유가 없음. 미디어 콘텐츠가 있으면 관심만 있어도 주기적으로 돌아옴.
02 · 새로운 방향

도구에서 미디어로

사용자가 목적을 갖고 찾아오는 "도구"에서, 관심 있는 사람이 즐겨 소비하는 "미디어"로 전환합니다. 데이터는 사라지지 않습니다 — 콘텐츠의 배경이 됩니다.

"호갱노노는 전국에서 창원 데이터를 봄. 우리는 창원 사람들이 읽고 싶은 창원 이야기를 만든다."

AS-IS · 도구형
검색 → 데이터 탐색
사용자가 찾고 싶은 게 생겼을 때만 방문. 비교·분석 기능이 메인. 오늘 집 살 계획이 없으면 올 이유 없음.
TO-BE · 미디어형
콘텐츠 → 데이터 연결
운영진이 SNS/카페에 콘텐츠를 배포 → 유입 → 관련 데이터로 자연 연결. 관심만 있어도 주기적으로 소비.
항목 호갱노노·리치고 네이버 카페 창부레터 (가안, 새 방향)
커버리지 전국 창원·김해 로컬 창원·김해 로컬
콘텐츠 없음 (데이터만) 있음 (글·댓글) 있음 (카드뉴스·기사)
구조화된 데이터 있음 없음 있음 (콘텐츠 연결)
SNS 배포 없음 부분적 핵심 채널
재방문 동기 집 살 때만 커뮤니티 활동 새 콘텐츠 소비
03 · 타겟 사용자

하나의 서비스가 모두를 만족시킬 수 없습니다. 메인 타겟을 명확히 하고, 나머지는 자연스럽게 따라오게 합니다.

04 · 포지셔닝

우리는 누구인가

서비스의 정체성을 한 문장으로 정의하면 모든 기능 결정과 디자인이 이 기준으로 판단됩니다.

창부레터는 네이버카페 창원부동산이야기 20만 회원의 신뢰를 바탕으로, 창원·김해 사람들이 우리 동네 부동산 이야기를 가장 먼저 접하는 공식 미디어입니다.

이런 기능/결정은 맞다 ✓
  • ✓ 주간 실거래가 카드뉴스
  • ✓ "이번 달 창원 가장 많이 거래된 단지"
  • ✓ 기획기사 "왜 중동 아파트가 올랐나"
  • ✓ 카드 → 단지 상세 데이터 연결
  • ✓ 알림 구독 (새 콘텐츠 + 단지 가격 변동)
이런 기능은 홈에서 뺀다 ✗
  • ✗ 투자분석 홈 노출 (탭 안으로)
  • ✗ 학원 추천 독립 섹션 (단지 상세 내)
  • ✗ AI 예측 배너 (기사 내 임베드)
  • ✗ 관리비 독립 메뉴 (단지 탭)
  • ✗ "Powered by AI" 강조 배지

06 · UX 레이어 구조

화면의 계층 구조

사용자가 어디서 유입되고, 어떻게 더 깊이 들어가는지의 레이어를 정의합니다.

발견층
(Discovery)
인스타그램 카드뉴스 네이버 카페 포스팅 구글 검색 (기획기사) 지인 공유 구독 이메일
→ 콘텐츠가 마케팅을 대신한다. 광고 없이 유입.
진입층
(Landing)
콘텐츠 상세 페이지 홈 피드 카드뉴스 풀 뷰 기획기사
→ SNS 링크 클릭 후 첫 화면. 콘텐츠가 완결성을 가져야 이탈 없음.
탐색층
(Explore)
관련 단지 데이터 임베드 단지 실거래가 상세 지역 가이드 관련 기사 추천 단지 비교
→ 콘텐츠에서 데이터로 자연스럽게 이동. "자세히 보기" CTA가 핵심.
전환층
(Convert)
가격 알림 구독 콘텐츠 알림 구독 즐겨찾기 단지 추가 광고주 문의
→ 한 번 구독하면 콘텐츠가 올라올 때마다 자동 재방문.
심화층
(Power)
갭투자 분석 AI 가격 예측 단지 비교 도구 학원 추천 관리비 조회
→ 기존에 만든 기능들. 홈 노출 없이 단지 상세 탭으로 접근.

11 · 다음 단계

무엇부터 시작하나

전체를 한번에 바꾸지 않습니다. 가장 적은 변경으로 가장 큰 방향 전환을 만드는 순서입니다.

1

콘텐츠 상세 페이지 신설

인스타/카페에서 링크 클릭 시 도달하는 랜딩 페이지. 카드뉴스 풀 뷰 + 관련 단지 데이터 임베드 + 구독 CTA. 이것이 없으면 퍼널이 완성되지 않음.

→ 이것 하나만 만들면 즉시 SNS 배포 효과가 생김
2

홈 피드형 재설계

검색창 중심 → 콘텐츠 피드 + 랭킹 스트립 + CTA. 네비게이션에 "콘텐츠" 추가. 지도·투자분석 메뉴는 depth 낮춤.

→ 기존 기능 삭제 없이 홈 레이아웃만 교체
3

기획기사 작성 툴 추가

어드민에 마크다운/리치텍스트 에디터 추가. 운영진이 기획기사를 직접 작성·발행. 데이터 임베드 단축키 지원.

→ 카드뉴스 빌더와 함께 핵심 제작 인프라 완성
4

지역 가이드 섹션 신설

"창원 처음이라면" 동별 가이드 페이지. 구글 SEO로 타지역 유입 창구. 삼성·포스코 임직원 이전 수요 타겟.

→ 기존에 없는 신규 콘텐츠 — 차별화 포인트
